# Mobile Sales Power BI Dashboard

## Overview
This project presents an interactive Power BI dashboard for analyzing mobile phone sales data. The dashboard provides insights into sales performance, customer behavior, payment methods, and product trends across different cities and time periods.

## Dashboard Preview
![Dashboard](dashboard.png)

## Key Insights
- Total sales, quantity, and transaction overview
- Sales distribution by city using map visualization
- Monthly sales trends
- Customer ratings analysis
- Transaction breakdown by payment method
- Brand and mobile model performance
- Sales trends by day of the week

## Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Dataset
The dataset includes mobile sales data with details such as brand, model, city, payment method, transaction count, quantity sold, and customer ratings.

## Conclusion
This dashboard helps stakeholders quickly understand sales performance and make data-driven decisions using interactive Power BI visuals.
